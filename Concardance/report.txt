для построения конкорданса был использован уже имеющийся лемматизатор текста
Алгоритм работы следующий:
    1. Загружается словарь
    2. Читается и с помощью словаря лемматизируется фраза а также читается окно для поиска контекста
    3. Построчно вычитывается и лемматизируется текст
    4. В лемматизированом тексте находятся вхождения фразы
    5. Рядом с каждым вхождением находятся левый и правый контексты
    6. Для подсчета статистики подсчитываются вхождения каждого контекста

Некоторые результаты:
самый часто встречаемый контекст для "российский":
российский рынок - 217
российская компания - 200 
на российский - 178
российский газ - 146

забавный результат получается для запроса "российский рынок"
на российский рынок - 101
с российский рынок - 41
уход с российский рынок -  32
